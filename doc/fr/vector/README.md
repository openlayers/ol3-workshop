# Faisons un éditeur d'objets géographiques!

Dans ce module, nous allons créer un éditeur de base pour travailler avec des données vecteur. Notre objectif est de faire en sorte qu'un utilisateur puisse importer des données, dessiner de nouveaux objets géographiques, modifier les objets géographiques existants et exporter le résultat. Nous travaillerons avec des données [GeoJSON](http://geojson.org/) dans ce module, mais OpenLayers prend en charge une large gamme de formats vectoriels si vous souhaitez travailler avec d'autres sources.
